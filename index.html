HTML

  • The Head element contains information about the webpage that the user cannot see such as metadata, which is used by the browser, search engines, and other developers behind the scenes.
  • The Body element contains the visible content of the webpage shown to the user. This includes tagged elements such as headings, paragraphs, ordered and unordered lists, images, anchor elements (which are used to create hyperlinks), and the line breaks that are sometimes placed between these elements.
  • The Footer element typically contains authorship information, copyright information, contact information, sitemap, and related documents.
  • Tags can have attributes within them. An HTML attribute defines additional information about an element, and are always defined in the beginning tag of an element, never the closing tag.

CSS

  • There are 3 types of CSS: Inline CSS, Internal CSS Style Sheet, and External CSS Style Sheet.
  • Inline CSS applies to a specific HTML tag from within the HTML file itself to style only a particular page element.
  • Internal Style Sheets are entire style sheets that are embedded directly into the HTML file using a Style element within the Head element. The rules only apply to that page.
  • External Style Sheets are a separate CSS file which is linked to the HTML file via a Link element. This is the most common method as it maintains Separation of Concerns.
  • Selector: Defines the element or atrributes to which rules or declarations will apply.
  • Declarations: Comprised of two components; property and value. Property is what we want to modify about the selector, and value is in what way we want to modify it. (Example "color: blue")
  • A margin indicates how much space we want around the outside of an element.
  • Padding indicates how much space we want around the content inside an element.

Git

  • pwd: Shows what current working directory you are in.
  • cd (directory name): moves you between nearby directories in gitbash command line
  • cd .. : Moves you back up one directory.
  • git status: checks what branch we are currently on
  • git checkout -b (branch name): creates a new branch and switches to it
  • git branch: Lets you see what branches are available.
  • git add -A: Commits all files in the current working branch.
  • git commit -m "associated message here": Commits changes to repository with a descriptive message about the changes.
  • git pull origin main: Pulls the base remote branch to ensure the current local branch is up to date.
  • touch (filename): Creates a file in the current directory.

JavaScript

  • Variables are named containers that allow us to store data.
  • Control flow is the order in which a computer executes code in a script. In JavaScript, code runs from top to bottom.
  • Conditional statements and loops can be used to control the order in which code is executed, or not executed.
  • If Statements are a type of conditional statement that will execute code only if the statement meets a condition, or is true.
  • The Strict Equality Operator (===) is used to check if two values are equal and returns a boolean result of true or false.
  • An Array is a single variable that is used to contain a set of multiple pieces of data, which are indexed in order starting from zero.
  • a For Loop is a type of loop which uses the predictable pattern of indices to perform a task repeatedly on all of the items in an array.
